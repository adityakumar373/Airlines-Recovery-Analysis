# Airline Recovery Analysis (2019–2025)

This project analyzes the recovery of major Indian and Gulf airlines after COVID-19 using publicly available airline disclosures.

## Objective
To compare airline recovery trends using key metrics:
- Passenger traffic
- Revenue (USD)
- Fleet size
- Network destinations

## Scope
Airlines analyzed:
- IndiGo
- Air India
- SpiceJet
- Emirates
- Qatar Airways
- Etihad Airways

Time period: 2019-2025

## Tools Used
- Microsoft Excel (data cleaning & normalization)
- Power BI (visual analytics)
- Public airline annual reports

## Key Analysis
- Recovery over time (2019–2025)
- Airline-to-airline comparison
- India vs Gulf airline comparison

## Files
- Airlines_Data.xlsx - cleaned dataset
- Summary.pdf - business insights
- PowerBI_Visuals.pdf - pdf of visual analysis
- Airlines.pbix - PowerBI files of visualizations
